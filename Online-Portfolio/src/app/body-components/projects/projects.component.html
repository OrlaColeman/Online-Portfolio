This is a quick description of this project. I created an online ordering system for kitchen staff This is instead of manually calling in every order, I have created an application where all companys are available to order from. Using this means all orders from meat to veg are stored in the same account and can be ordered using just one application. This is to help reduce errors including missing items or ordering too much. A major issue in kitchens is orders and finding the time to do them. With this application it should be easy and quick to do, also allowing for staff who don't usually make orderes able to do so by just looking at previous orders. It also facilitates data analysis of what is being spent and where savings can be made with is all in one space.
